# Power-BI-Sales-Customer-Analytics-Project
This project presents an end-to-end sales analytics dashboard built using Power BI. It includes data modeling with star schema, DAX-based calculations, time intelligence analysis, and interactive insights on sales, profit, and returns across regions, products, and customers.
